Literature Review 8

Project Name: A Secure and Scalable Online Voting System

Summary: This paper presents a secure online voting system built using the MERN stack (MongoDB, Express.js, ReactJS, Node.js), addressing concerns such as data protection, authentication, and resilience against attacks.

Key Outcomes:

  • Secure online voting system using MERN stack
  • Comprehensive data protection mechanisms
  • Robust authentication and authorization systems
  • Resilience against various cyber attacks
  • Scalable architecture for large-scale voting
  • Modern full-stack development implementation

Technical Features:

  • MongoDB for flexible data storage
  • Express.js for backend API development
  • ReactJS for dynamic frontend interface
  • Node.js for server-side JavaScript runtime
  • Advanced security protocols and encryption
